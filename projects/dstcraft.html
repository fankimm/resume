주요 기능

크래프팅 레시피 검색

모든 제작 레시피, 재료, 제작소를 검색하고 탐색. 캐릭터별 전용 아이템 필터링, 카테고리별 분류.

Crock Pot 시뮬레이터

최대 4개 재료를 조합하여 요리 결과를 미리 확인. 실제 게임 로직을 기반으로 한 정확한 결과 예측.

보스 데이터베이스

시즌별, 레이드, 해양, 던전, 이벤트 보스의 체력, 공격 패턴, 드랍 아이템 정보 제공.

다국어 지원 (i18n)

영어, 한국어, 일본어, 중국어, 프랑스어, 독일어, 스페인어, 러시아어 등 13개 언어 지원. 게임 내 아이템명도 현지화.

PWA

Service Worker 기반의 오프라인 지원. 모바일/데스크톱에서 앱처럼 설치 가능.

기술적 챌린지

Problem

게임 데이터가 공식 API 없이 Lua 소스코드에만 존재. 수백 개의 아이템/레시피 데이터를 구조화해야 함.

Solution

게임 소스의 Lua 파일을 분석하여 데이터 구조를 파악하고, TypeScript 타입 시스템으로 정적 데이터셋 구축. 카테고리, 태그, 관계 데이터를 체계적으로 정규화.

Result

100개 이상의 아이템과 레시피를 타입 안전하게 관리. 게임 업데이트 시 데이터 추가가 용이한 구조.

Problem

SEO를 통한 검색 유입 확보 필요. 하지만 SPA 구조는 크롤러 친화적이지 않음.

Solution

Next.js 15의 Static Export를 활용한 완전 정적 사이트 생성. 각 아이템/레시피별 개별 페이지와 메타데이터 자동 생성으로 크롤러 최적화.

Result

게임 관련 키워드로 검색 유입 확보. Vercel 호스팅으로 빠른 로딩 속도 유지.
