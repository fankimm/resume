PILS

2025.09 ~ 2026.03 퍼플아이오 (코오롱FnC 자회사)

개인정보 내부관리 시스템(Personal Information Lifecycle System). 개인정보 접근 이력 감사, 과다 접근 모니터링, 소명 프로세스, 컴플라이언스 리포트를 제공하는 사내 보안 관리 플랫폼.

Next.jsNextAuthTailwindCSS PlaywrightTipTapTypeScript

미리보기


📸 PILS 대시보드 또는 감사 로그 화면 스크린샷

담당 역할

  • 프로젝트 프론트엔드 전체 설계 및 단독 개발
  • NextAuth 기반 사내 인증 시스템 구축 (세션 관리, 권한 분리)
  • TipTap 에디터 기반 소명 게시판 개발
  • Playwright E2E 테스트 작성으로 핵심 플로우 자동 검증
  • 개인정보 접근 행위 감사 로그 시각화 및 필터링 UI

주요 기능

접근 행위 감사

사용자별 개인정보 접근 이력을 실시간으로 기록하고 시각화. 접근 시간, 대상 데이터, 접근 사유를 추적하며 비정상 접근 패턴 감지.

과다 접근 모니터링

설정된 임계치를 초과하는 개인정보 조회를 자동 탐지. 알림 발생 시 담당자에게 즉시 통보하고 소명 절차 시작.

소명 게시판

TipTap 리치 텍스트 에디터 기반의 소명 작성 및 관리 시스템. 소명 요청 → 작성 → 검토 → 승인/반려 워크플로우.

컴플라이언스 리포트

기간별 개인정보 접근 통계, 과다 접근 발생 현황, 소명 처리율 등을 집계한 컴플라이언스 보고서 자동 생성.

기술적 챌린지

Problem

사내 보안 요구사항에 맞는 인증/인가 시스템 필요. 일반 사용자, 관리자, 감사자 등 역할별 접근 제어가 필수.

Solution

NextAuth를 활용한 세션 기반 인증과 미들웨어 레벨의 역할별 라우트 가드 구현. 서버 컴포넌트에서의 세션 검증으로 보안 강화.

Result

역할별 완전한 접근 분리 달성. 인증되지 않은 접근 시도 0건 유지.

Problem

보안 관련 시스템 특성상 버그로 인한 데이터 누락이나 잘못된 감사 기록이 발생하면 컴플라이언스 위반으로 이어질 수 있음

Solution

Playwright를 활용한 E2E 테스트로 핵심 플로우(로그인 → 감사 로그 조회 → 소명 작성 → 리포트 생성) 자동 검증 구축

Result

배포 전 자동화된 품질 검증으로 프로덕션 버그 최소화. 테스트 커버리지 91% 달성.
