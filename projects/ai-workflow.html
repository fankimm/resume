2025 ~ 현재 퍼플아이오 (코오롱FnC 자회사)

Claude Code와 MCP(Model Context Protocol)를 활용해 개발 전 과정을 자동화한 워크플로우. 태스크 할당부터 코드 구현, 코드 리뷰, 배포, 알림까지 사람 개입을 최소화.

워크플로우

Asana 폴링 → 태스크 분석 → Claude Code 구현 → GitLab MR → CI/CD 배포 → Slack 알림

📸 Slack 알림 스크린샷 또는 워크플로우 실행 화면

구성 요소

Asana MCP 연동

Asana 프로젝트의 태스크를 주기적으로 폴링하여 새로운 작업 감지. 태스크 제목, 설명, 우선순위를 파싱하여 개발 컨텍스트로 변환.

Claude Code 자동 구현

파싱된 태스크 컨텍스트를 기반으로 Claude Code가 코드 분석 → 구현 → 테스트까지 자동 수행. MCP를 통해 프로젝트 구조와 코드베이스 컨텍스트를 제공.

GitLab MR & CI/CD

구현 완료 시 자동으로 GitLab Merge Request 생성. CI 파이프라인 통과 후 스테이징/프로덕션 배포까지 자동 진행.

Slack 알림

워크플로우 각 단계의 진행 상황과 결과를 Slack 채널로 실시간 알림. 에러 발생 시 즉시 담당자에게 멘션.

성과

  • 단순 반복 개발 태스크의 처리 시간을 대폭 단축
  • 야간/주말에도 태스크 자동 처리로 개발 병목 해소
  • 코드 리뷰 전 자동화된 품질 검증으로 리뷰 부담 감소
  • Asana → 코드 → 배포 → 알림까지 엔드투엔드 자동화 달성
  • 팀 내 AI 기반 개발 프로세스 도입의 선례 확립
