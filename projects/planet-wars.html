게임 특징

  • p5.js Canvas 기반의 실시간 2D 렌더링
  • 행성 간 유닛 이동과 전투 시뮬레이션
  • AI 적군의 전략적 의사결정 로직
  • 파티클 이펙트와 시각적 피드백
  • GitHub Pages로 배포하여 브라우저에서 바로 플레이 가능

구현 포인트

게임 루프 & 물리

p5.js의 draw() 루프를 활용한 60fps 게임 루프. 유닛 이동 시 행성 간 거리, 속도, 충돌 판정을 계산하는 간단한 물리 시뮬레이션.

AI 로직

적 AI는 현재 전력 비교, 가장 약한 행성 타겟팅, 방어/공격 전환 등의 규칙 기반 의사결정으로 플레이어에게 도전감을 제공.

BONUS PROJECT

Wave Synth

Web Audio API를 활용한 브라우저 기반 신디사이저. 음악과 코드의 접점을 보여주는 크리에이티브 코딩 프로젝트로, 다양한 파형(Sine, Square, Sawtooth, Triangle)을 조합해 소리를 생성.

Web Audio APIJavaScriptAudioContextOscillator

주요 기능

  • 키보드 입력으로 실시간 음 생성 (피아노 키 매핑)
  • 파형 선택: Sine, Square, Sawtooth, Triangle
  • ADSR 엔벨로프 컨트롤 (Attack, Decay, Sustain, Release)
  • Web Audio API의 AudioContext, OscillatorNode, GainNode 활용
